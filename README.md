# electron-websocket-chat

Electron websocket聊天演示，一个简单的DEMO，从浏览器端发送消息到Electron客户端。

## 安装

```
npm install
```

## 运行

### 启动server

```
node server/index.js
```

### 在浏览器中启动第一个客户端

```
file:///path/electron-websocket-chat/index.html
```

### 启动electron客户端

```
npm run start
```

### 在浏览器客户端中输入任意字符

Enjoy it!

### 打包

```
npm run package
```
